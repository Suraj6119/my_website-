by default, links will appear as follows in all browsers :-


  • An unvisited link is underline and blue.
  • A visited link is underline and purple.
  • An active link is underline and red.

    • HTML Links :- The target attributes

    The target attribute can have one of the following values :-


  • _self :- Default. Opens the documents in the same window/tab as it was clicked

    • Example

    Learn Machine Learning visit, Machine Learning

  • _blank :- Opens the document in a new tab or window

    • Example

    learn C++ language, visit Learn C++

  • _parent :- Openes the document in the parent frame

    • Example

    Learn everything, visit GFG

  • _top :- Openes the document in the full body of the window

    • Example

    learn Python, visit python.org

    A perfect web address is a Absolute URL.

    A local(link to a page within the same website) is specified with a Relative URL.

    Example

    A Absolute URL of official Python website, python.org.

    A Relative URL of Python.org Python tutorial.
